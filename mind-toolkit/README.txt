== Description ==

Mind Toolkit WordPress plugin can help you create templates for your website. TheÂ plugin offers extra-ordinary features, including easy-to-use options for creating and customizing templates for ultimate needs. It is compatible with elementor website builder.

== Installation ==

= From the dashboard: =
1. Go to the WordPress Dashboard "Add New Plugin" section
2. Search For "Mind Toolkit"
3. Install, then Activate it

= From the control panel: =
1. Unzip (if it is zipped) and Upload `mind-toolkit` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
